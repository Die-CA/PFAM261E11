Descripcion:
Aplicación móvil desarrollada en Kotlin + Jetpack Compose, diseñada para gestionar tareas diarias, agregar recordatorios locales, capturar imágenes desde la cámara con CameraX, y programar notificaciones personalizadas.

Características:
-Crear o eliminar tarea.
-Programacion de notificaciones (próximamente).
-Visualizar tareas.
-Agregar imagen desde la galeria o tomar imagen desde la app.
-Pantalla de configuración.

Tecnologias usadas:
-Kotlin
-Jetpack Compose
-Android ViewModel
-Room Database
-Alarm manager
-CameraX
-DataStore Preferences
-Material3

Instalación
Instalar desde APK
  1. Descarga el archivo app-release.apk (si lo agregas al repo o a Releases).
  2. Habilita Instalar apps desconocidas en tu dispositivo.
  3. Instala y abre la aplicación.
